Online Notiwire
=========

API for various services relevant for students in Trondheim.

Running on notiwire.online.ntnu.no

https://notiwire.online.ntnu.no/api/affiliation/online

https://notiwire.online.ntnu.no/api/cantina/realfag

## Setup

Install mongodb

```javascript

npm run init
npm start
```

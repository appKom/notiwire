Online Notiwire
=========

API for various services relevant for students in Trondheim.

Running on passoa.online.ntnu.no

http://passoa.online.ntnu.no/api/affiliation/online

http://passoa.online.ntnu.no/api/cantina/realfag

## Setup

Install mongodb

```javascript
npm install
npm run init
npm start
```
